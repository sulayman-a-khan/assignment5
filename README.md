1. Difference between var, let and const

var: 	  Old way. Works in the whole function. Can be changed or declared again.
let: 	  Modern way. Works only inside the block. Can be changed.
const:  Most useful variable. Works only inside the block. Cannot be changed or redeclared.


2. What is the spread operator?
We can spread elements using this. For example, inside an array if we want to add some items, we can use the spread operator to merge them.


3. What is the difference between map(), filter() and forEach()?

map():
Works on every element of an array and returns a new array.
It is mainly used to modify or transform elements.

filter():
Checks every element of an array against a condition and returns a new array with the elements that pass the condition.

forEach():
Runs a loop through each element of an array but doesn’t return a new array.


4. What is an arrow function?
An arrow function is a shorter way to write a function using the => arrow syntax.

5. What are template literals?
Template literals are a way to create strings using backticks.
We can write multiline strings or make it dynamic as needed.
